## [1.0.0] - 2026-02-17

### Features
- **Calendar View**: Visual overview of daily alcohol consumption with "liver days" indicators.
- **Record List**: Detailed timeline of drinks with edit and delete capabilities.
- **Analytics**: Visualization of total alcohol and calorie intake trends.
- **Custom Settings**: Configurable drink types, alcohol percentages, and calorie coefficients.
- **Google Sheets Sync**: Secure data backup and synchronization using Google Apps Script.
- **PWA Support**: Installable on mobile devices for a native app-like experience.
- **Dark Mode**: Optimized UI for low-light environments.
